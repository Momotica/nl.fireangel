# FireAngel CO Sensor support for Athom Homey  
Makes Homey work with the Z-Wave version of the FireAngel CO Sensor.
   
## Supported devices with most common parameters:
* FireAngel CO Sensor NM-CO-10X-INT
   
## Supported Languages:
* English
* Dutch
   
## Change Log:

### v 1.0.0
